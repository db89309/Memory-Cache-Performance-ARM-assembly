# Memory-Cache-Performance-ARM-assembly
source code for Parts 1 &amp; 2


Both files were made using CPUlator for ARMv7 and MIPS.
You must select the right language before uploading the docs to use the source code.
Once the corresponding code has been selected, simply upload the code by going to File >> Open >> and selecting it within your files
Then simply click "Compile and Load" and then "Continue" which should prompt the corresponding questions for the assignment in the JTAG UART.
